In previous prediction tasks, sample points were acquired via real-time online collection. For Task 1, this has been updated to use the provided fixed sample points. Additionally, the sample locations for Task 2 have been submitted for review (available in dataset/sample). Since these sample points are binary, you will need to place the labels for B1 and B5 in the directory dataset/Outputs/Task\_1\_ICASSP. These will be multiplied by the binary sample points to generate sample channels with signal strength information and to calculate errors against the predicted maps. Finally, execute the script (e.g., test\_image\_002task1.py) to complete the prediction; results will be saved in the image\_result folder.
